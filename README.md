# duplicate-issue-checker

## Tech Stack
- **Language**: Python 3.10
- **Framework**: Flask
- **Frontend**: TailwindCSS

## Getting Start
1. ```sh
   python -m venv venv
   ```
2. （for Windows）  
   ```sh
   .\venv\Scripts\activate
   ```
3. ```sh
   pip install -r requirements.txt
   ```
4. ```sh
   python .\run.py
   ```
5. ```sh
   npm install
   ```
6. ```sh
   npm run start
   ```
7. Access http://localhost:5000/

## Run test
```sh
pytest
```

## Run lint
```sh
pylint app/**/*.py tests/**/*.py run.py
```
